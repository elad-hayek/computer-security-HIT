# Communication Ltd - Secure vs Vulnerable Project

## Setup

Run the commands inside either **communication-ltd-secure** or **communication-ltd-vulnerable**:

```bash
cd communication-ltd-secure
npm install
npm run db:init or db:reset
npm run dev
```
Create **.env.local** file in each project with the following keys:

```
DB_PATH=./data/communication_ltd.db
GMAIL_USER=
GMAIL_APP_PASSWORD=
GMAIL_FROM_EMAIL=
```
